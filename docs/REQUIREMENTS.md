# Daytripper
## User Stories
1. I would like to be able to see when the next train is coming from my station to my destination.
2. I would like to be able to see if I need to transfer to another train to get to my destination.
3. I would like to be able to see the estimated time of arrival:
    - At my station for the next train to my destination.
    - At my destination for the next train from my station.
    - Updated in real-time as the train moves along the route.
    - Even if I am not using the app
4. I would like to have a home station so that creating trips is easier.
5. I would like to be able to delete saved trips from my account.
6. I would like to be able to add trips to folders for organization.
7. I would like to be able to see the estimated time of arrival for my train at any station along the route, even if I am not currently at that station, and have that information be updated in real-time.
8. I would like this app to be available on both iOS and Android so that I can use it regardless of my device.
9. I would like to be able to create a new trip by selecting a station on the map and then selecting a destination station, rather than having to type in the station names.
10. I would also like the option of typing in the station names if I prefer that method of input.
